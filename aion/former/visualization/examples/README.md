# aion.former.visualization — Examples

## Run

```bash
pip install -e ".[former]"
python -m aion.former.visualization.examples.demo_attention_plot
```

Writes **`attention_demo.png`** in the current working directory (non-interactive **Agg** backend).

| Script | Content |
|--------|---------|
| **demo_attention_plot.py** | Synthetic attention weights → `plot_attention_map` → PNG. |
